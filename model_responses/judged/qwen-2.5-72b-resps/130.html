To find the common elements in two arrays without using any extra data structures, you can use a simple approach that involves sorting and then using two pointers to traverse both arrays. Here's a step-by-step explanation and a Python implementation of this approach:

Step-by-Step Explanation

  1. Sort Both Arrays: First, sort both arrays. This will allow us to use a two-pointer technique to find common elements efficiently.
  2. Initialize Two Pointers: Use two pointers, one for each array, starting at the beginning of each array.
  3. Traverse Both Arrays:
    • Compare the elements at the current positions of the two pointers.
    • If the elements are equal, it means we have found a common element. Add it to the result list and move both pointers forward.
    • If the element in the first array is smaller, move the pointer in the first array forward.
    • If the element in the second array is smaller, move the pointer in the second array forward.
  4. Continue Until One Array is Fully Traversed: The process continues until one of the pointers reaches the end of its array.

Python Implementation

```python def findcommonelements(arr1, arr2): # Step 1: Sort both arrays arr1.sort() arr2.sort()

# Step 2: Initialize two pointers
i = 0
j = 0

# Step 3: Initialize a list to store common elements
common_elements = []

# Step 4: Traverse both arrays
while i < len(arr1) and j < len(arr2):
    if arr1[i] == arr2[j]:
        # Found a common element
        common_elements.append(arr1[i])
        i += 1
        j += 1
    elif arr1[i] < arr2[j]:
        # Move the pointer in the first array forward
        i += 1
    else:
        # Move the pointer in the second array forward
        j += 1

return common_elements

Example usage

arr1 = [1, 3, 4, 5, 7] arr2 = [2, 3, 5, 6] print(findcommonelements(arr1, arr2)) # Output: [3, 5] ```

Explanation of the Code

  • Sorting: The sort() method is used to sort both arrays in ascending order.
  • Two Pointers: Two pointers i and j are initialized to 0, pointing to the start of arr1 and arr2, respectively.
  • Traversal: The while loop continues as long as neither pointer has reached the end of its respective array.
    • If the elements at arr1[i] and arr2[j] are equal, they are added to the common_elements list, and both pointers are incremented.
    • If arr1[i] is less than arr2[j], the pointer i is incremented to move to the next element in arr1.
    • If arr1[i] is greater than arr2[j], the pointer j is incremented to move to the next element in arr2.

This approach ensures that we find all common elements without using any additional data structures, and it runs in O(n log n + m log m) time due to the sorting step, where n and m are the lengths of the two arrays. The space complexity is O(1) since no extra data structures are used.
